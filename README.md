# React + Tailwind Starter Pack 🚀

A modern starter template built with:

- ⚛️ React
- ⚡ Vite
- 🎨 Tailwind CSS v4

Perfect for quickly starting frontend projects with a clean and minimal setup.

---

# 📦 Features

- React + Vite setup
- Tailwind CSS configured
- Clean folder structure
- Fast development server
- Ready-to-use starter template

---

# 🚀 Installation & Setup

## 1. Clone the Repository

```bash
git clone https://github.com/Zephyrex21/React_Tailwind_Starter_Pack.git
```

---

## 2. Move Into Project Folder

```bash
cd react-tailwind-starter-pack
```

---

## 3. Install Dependencies

```bash
npm install
```

This will automatically install all required packages from `package.json`.

---

## 4. Start Development Server

```bash
npm run dev
```

---

# 🌐 Open in Browser

After running the dev server, open:

```bash
http://localhost:5173
```
